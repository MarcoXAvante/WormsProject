# Controls:

- A: Move Right

- D: Move Left

- Aim with your mouse position

- SPACE: Shoot

- R: Skip Turn

- F1: Enable/Disable Debug

- F2: Change mode

- F3: Collision type

- F4: Enable/Disable Aerodynamics

- F5: Enable/Disable Hydrodynamics
